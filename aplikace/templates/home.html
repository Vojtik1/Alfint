About Alfint

Alfint is designed to help investors make smarter decisions by providing in-depth analytics and backtesting tools based on historical and real-time stock data. We use cutting-edge technologies like Python, Django, and various APIs to offer a seamless, powerful experience for investors looking to optimize their portfolios.

Technologies We Use

Python

Python is the backbone of our platform. We use it for data processing, analysis, and backend services, enabling a smooth and efficient workflow.

Django

Django powers the backend of Alfint, ensuring robust performance and scalability. It offers a secure framework for building the web application.

SimFin API

SimFin provides us with comprehensive financial data, including key stock fundamentals and historical data for in-depth analysis.

yFinance API

yFinance offers real-time stock data and financial information, allowing us to integrate live metrics and up-to-date market insights into our platform.

JavaScript

JavaScript enhances the frontend of our application, enabling dynamic features such as interactive charts, graphs, and real-time data updates.
